# 專案工作規則

## 👤 使用者溝通偏好（每次回覆都要照做）

使用者不懂程式碼。回覆指示時必須明確標示「對象」，不然使用者不知道這段話是要給誰的：

- **👉 貼給 Claude Code**：要使用者複製貼到 Claude Code 終端機跑的指令 / 對話（用 code block 包起來）
- **🖐 你手動**：使用者要在瀏覽器 / Zeabur / Notion / 其他 GUI 手動操作的事
- **其他一般說明**：正常回文字就好

範例：
> 🖐 你手動：到 Zeabur Dashboard → real-estate-crm → 環境變數 → 新增 `DATABASE_URL` = `${postgresql.DATABASE_URL}`
>
> 👉 貼給 Claude Code：
> ```
> 執行 mba skill，開始 Step 5b
> ```
