QUICK START
================================================================================

1. INSTALL:
   npm install

2. CONFIGURE:
   cp .env.example .env.local
   # Add your Notion API Key and database IDs

3. RUN:
   npm run dev
   # Visit http://localhost:3000

4. BUILD:
   npm run build
   npm start

================================================================================
NOTION DATABASE SCHEMA
================================================================================

Required Databases:

1. 買家 (Buyers)
   - 名稱 (Title), 手機, NOTE, 最近進展
   - 等級 (Select), 來源, 預算, 需求, 區域
   - 下次跟進 (Date)

2. 任務 (Tasks)
   - 名稱 (Title), 完成 (Checkbox), 客戶 (Relation)

3. 追蹤與委託 (Tracking)
   - 名稱 (Title), 地址, 價格, 狀態

4. 週報 (Weekly)
   - 日期 (Date), 看房次數 (Number), 進展摘要

5. 策略 (Strategy)
   - 名稱 (Title), 狀態, 瀏覽次數

6. AI想法 (AI Ideas)
   - 名稱 (Title), 狀態, 平台 (Multi-select)

API ENDPOINTS
================================================================================

GET   /api/clients                    - Fetch all clients
PATCH /api/clients/[id]              - Update client
GET   /api/properties                - Fetch properties
GET   /api/weekly                    - Fetch weekly data
GET   /api/todos                     - Fetch tasks
GET   /api/videos                    - Fetch videos
GET   /api/ai-projects              - Fetch AI projects
GET   /api/setup                     - Check setup status

CUSTOMIZATION GUIDE
================================================================================

MODIFY COLORS/THEME:

MODIFY DATABASE FIELDS:
→ lib/notion.ts (type interfaces)

→ components/*.tsx (UI components)

MODIFY NAVIGATION:
→ components/Navigation.tsx (tabs array)

MODIFY PAGE LAYOUT:
→ app/*/page.tsx (individual pages)
→ app/globals.css (global styles)
